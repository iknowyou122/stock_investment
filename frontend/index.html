分點行為分類

四種分點行為，
一眼看穿籌碼意圖

每個分點代碼背後，都有一套可量化的歷史行為模式。我們的分類器將歷史數據轉化為可直接使用的標籤。

隔日沖
隔日沖型主力

買超後隔日高機率反手賣出，識別短線籌碼。D+2 反轉率超過 74%，是籌碼最危險的警示信號。看到這個標籤出現在前三名，立刻提高警覺。

波段贏家
波段主力

持股周期 3–15 日，識別具方向性的波段主力。歷史上，這類分點出現在前三名時，後續三日正報酬機率顯著高於基準。

地緣券商
地方性主力

特定區域集中交易，常見於地方性主力佈局。這類分點的交易行為往往與上市公司所在地的產業動態高度相關。

代操官股
機構投資人

低調持續買超，識別機構投資人底部吸籌。這類分點通常不追高，出現連續三日以上的買超行為具有重要意義。

數據說話，不是故事

所有標籤基於歷史實際成交紀錄，最少 50 筆樣本才會賦予分類。沒有猜測，沒有黑盒子。

python 
import requests

BASE = "https://api.fenzhi.tw"
headers = {"X-API-Key": "YOUR_KEY"}

# 查詢 凱基-台北 分點標籤
r = requests.get(
    f"{BASE}/v1/broker-label/9600",
    headers=headers
)
data = r.json()
print(data["label"])          # 隔日沖
print(data["reversal_rate"]) # 0.74

# 查詢 2330 信號
sig = requests.get(
    f"{BASE}/v1/signal/2330",
    headers=headers
).json()
print(sig["signal"])      # LONG
print(sig["confidence"]) # 88
javascript 
const BASE = "https://api.fenzhi.tw";
const KEY  = "YOUR_KEY";

async function getBrokerLabel(code) {
  const res = await fetch(
    `${BASE}/v1/broker-label/${code}`,
    { headers: { "X-API-Key": KEY } }
  );
  return res.json();
}

const label = await getBrokerLabel("9600");
console.log(label.branch_name);  // 凱基-台北
console.log(label.label);       // 隔日沖
