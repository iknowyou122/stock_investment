# 分析因子完整說明書

> 適用版本：Tier A 擴充後（2026-03-27）
> 對象：使用本系統的台股自主操盤手

---

## 總覽：三柱確認系統

系統用三個獨立維度評分，三柱都強才給 LONG 信號：

```
Pillar 1 動能   ── 價格和成交量的行為（個股本身）
Pillar 2 籌碼   ── 誰在買？聰明錢還是散戶？（資金結構）
Pillar 3 空間   ── 突破了嗎？趨勢結構是否有利？（位置與趨勢）
```

三柱缺一，信號就會被降級。這是設計上的刻意選擇：動能強但籌碼是隔日沖，不算真突破；籌碼好但空間壓力大，進場成本高。

---

## Pillar 1 — 動能（max 55 pts）

### 1-1. VWAP 5日均 +20

**為什麼**
5日成交量加權均價（VWAP）是法人常用的成本基準。收盤站上 VWAP，代表今天的主動買方願意用高於平均成本的價格持有部位。這比單純「漲」更有意義——成交量加權，所以大量在低點成交反而會把 VWAP 拉低，今收高 VWAP 代表買方控盤。

**量測什麼**
計算最近5個交易日每日 VWAP（= 當日成交額 / 成交量），取算術平均。今日收盤 > 這個5日平均 VWAP → 得分。

**怎麼用**
這是 Pillar 1 最重要的20分。沒有這分，後續其他動能分數意義有限。反過來說，如果 VWAP 得分但其他動能都沒有，可能只是今天短暫拉升。

---

### 1-2. 量能突破 +20

**為什麼**
量是價的先行指標。主力建倉必須透過成交量，無量的漲不可靠。但「量大」本身不夠——量大收黑（分配日）其實是賣出信號。這個因子加了方向條件：放量且收紅（或平），才給分。

**量測什麼**
今日成交量 > 20日均量 × 1.5，且今日收盤 ≥ 昨日收盤。兩個條件缺一不可。

**怎麼用**
量能突破是最直觀的主力介入信號。如果股票同時有 VWAP 站上和量能突破（共 +40），幾乎可以確定有機構力量。缺少量能突破但其他分數高，要小心是無量拉升，可能是假突破。

---

### 1-3. K線收盤強弱比 +5

**為什麼**
一根K線的形狀說明了買賣方在這段時間的控盤結果。收盤靠近日內高點，代表買方在尾盤仍有力量；收盤靠近低點（長上影線），代表漲上去後被賣壓打下來。

**量測什麼**
(收盤 - 最低) / (最高 - 最低) > 0.7。收盤在日內波動幅度上方30%以內才得分。如果 高=低（平盤整日），跳過（避免除以零）。

**怎麼用**
這5分是對K線品質的加分，不是主要依據。搭配量能突破使用：量大且收在高點，比量大但收在低點可靠得多。孤立的強K線強弱比意義有限。

---

### 1-4. 連漲天數 +5

**為什麼**
趨勢的慣性。連續收高代表每一個交易日結束時買方都略佔上風，這是短期動能持續的條件。3日連漲不是隨機事件——在多空均等的市場，連漲3日的機率約12.5%，通常反映持倉意願。

**量測什麼**
包含今日在內，連續幾個交易日收盤價逐日高於前一日。≥ 3日 → 得分。

**怎麼用**
這分搭配 VWAP 和量能使用。連漲本身是「持續性」確認，而非「突破」確認。如果三天都是無量小漲，連漲分拿到了但量能沒有，整體動能得分仍低。

---

### 1-5. 量能遞增趨勢 +5

**為什麼**
主力建倉通常是逐日加碼，不是單日爆量後消失。量能遞增（前3日成交量連續放大）代表市場參與度在提升，不是一次性脈衝。這與「今天量大」（1-2）是不同的量測——量趨勢 vs 量水位。

**量測什麼**
今日收盤前的3個完整交易日（T-3, T-2, T-1），成交量 T-3 < T-2 < T-1 三日連續遞增。注意：不含今日。

**怎麼用**
是量能突破的「前情提要」驗證。量能遞增後今日再放大突破，比突然爆量更可信。如果今天量突然爆，但前三天都在縮量，要注意是否是短線洗盤後的反彈。

## Pillar 2 — 籌碼（付費版 max 45 pts，免費版 max 63 pts）

> **付費版（FinMind TaiwanStockBrokerTradingStatement）** 和 **免費版（TWSE opendata）** 互斥。系統優先使用付費版；付費資料無法取得時自動切換免費版。

---

### 付費版因子（FinMind 分點資料）

---

### 2P-1. 淨買超家數差 +15

**為什麼**
分點（券商營業處）數量反映籌碼的廣度。一隻股票若有50個分點在買、只有10個在賣，代表廣泛的買方興趣，不是單一主力拉抬。這比「誰買最多」更難造假。

**量測什麼**
計算最近3個交易日的累積值：每日（有淨買超的分點數 - 有淨賣超的分點數）加總。如果3日累積值 > 0 → 得分。

**怎麼用**
這是付費版最重要的15分。淨買超家數差為正，代表市場廣度支持這個漲勢。與集中度（2P-2）配合：廣度大但集中度低，可能是散戶跟風；廣度大且集中度也高，最強訊號。

---

### 2P-2. 籌碼集中度 Top15 +15

**為什麼**
主力建倉的特徵是「集中」——少數幾個大型分點（通常是法人下單的大型券商）買入大量。如果買盤分散在100個小分點，可能是散戶跟進，不是主力建倉。

**量測什麼**
前15名買超分點的買入量 / 全市場總買入量 > 35%，且今日活躍分點（有買入）≥ 10家。第二個條件避免流動性極低的股票被誤判（只有3個分點交易，集中度100%但毫無意義）。

**怎麼用**
35%門檻是基於實踐的校準值。主板大型股（如台積電）的籌碼天然分散，35%是相對高集中；小型股35%則不算特別集中。這個因子在中型股（成交量10-50億）最有效。

---

### 2P-3. 前三無隔日沖 +10

**為什麼**
隔日沖（day-trade）主力是當天買、隔天賣。如果今天買最多的前3個分點歷史上都是「買進後隔天就賣」的行為模式，今天的買入根本不是持倉意願，明天就會成為賣壓。這是所有因子中最重要的品質過濾。

**量測什麼**
查詢 Top3 淨買超分點的 `label`。如果全部不是「隔日沖」標籤 → 得 +10。如果任何一個是隔日沖 → 不得 +10，反而扣 -25（見風險扣分）。

**怎麼用**
這個因子是門票概念。在付費版中，有隔日沖在 Top3 時，整個 Pillar 2 不只得不到分，還被扣 25 分。實際使用時，遇到 CAUTION 信號，先看 flags 是否有 `隔日沖_TOP3`。有的話，當天的漲幅很可能在明天被消化。

---

### 2P-4. 知名外資分點偵測 +5（新）

**為什麼**
台灣主要外資券商使用固定的分點代碼。摩根大通（1480）、美林（1560）、瑞銀（9200）等出現在買超前幾名，代表國際法人真實建倉，不是本地主力拉抬。外資買入通常有更長的持有期。

**量測什麼**
掃描 `top_buyers` 列表，比對8個已知外資分點代碼：
- 1480 摩根大通 / 1560 美林 / 9200 瑞銀 / 1770 花旗
- 2030 高盛 / 1710 法國巴黎銀行 / 8150 德意志 / 1790 麥格理

任一出現在買超名單 → +5。

**怎麼用**
外資分點出現是加分項目，不是必要條件。搭配集中度高（2P-2）使用：外資大量集中買入，是籌碼品質的最高確認。如果 flags 顯示 `FII_PRESENT: 摩根大通`，值得特別注意。

### 免費版因子（TWSE opendata）

> 以下因子在付費分點資料無法取得時啟動。資料來源：TWSE T86（三大法人）與 MI_MARGN（融資融券）。

---

### 2F-1. 外資買賣超 +15

**為什麼**
外資（外陸資）是台股最大的機構力量，持股比率常達 50% 以上（台積電超過 70%）。外資淨買代表國際資金認可當前估值，具有較強的持倉延續性。

**量測什麼**
TWSE T86 揭露的外陸資買賣超股數 > 0 → +15。

**怎麼用**
這是免費版中最重的15分。外資淨買當日，搭配動能突破，是最常見的短線強勢格局。注意：外資資料是全市場公告，當日交易所收盤後約1-2小時更新。

---

### 2F-2. 投信買賣超 +10

**為什麼**
投信（本地基金）是台股第二大法人。投信的買賣行為有明顯的月底作帳、季底作帳規律，但持倉周期相對較長（幾週到幾個月）。投信淨買通常是中期趨勢信號，不是當日進出。

**量測什麼**
TWSE T86 投信買賣超股數 > 0 → +10。

**怎麼用**
投信連買（2F-7）比單日投信買更有意義。單日投信買可能是月底作帳；連買3日以上才代表真實建倉意願。

---

### 2F-3. 自營商買賣超 +5

**為什麼**
自營商（券商自有資金）通常是短線操作，但自營商淨買仍代表機構判斷今日偏多。三大法人同向時（外資+投信+自營商全買），最弱的連自營商也加入，可信度最高。

**量測什麼**
TWSE T86 自營商買賣超股數 > 0 → +5。

**怎麼用**
孤立的自營商買超意義有限（分數也最低）。主要功能是觸發「三大法人同向」加分（2F-5）。

---

### 2F-4. 融資餘額變化 +10

**為什麼**
融資（信用買進）代表散戶用槓桿追漲。融資增加通常是散戶跟風，不是法人建倉——因為法人用自有資金或期貨，不用融資。當一隻股票在漲但融資餘額沒有增加（甚至減少），代表漲是法人拉抬而非融資散戶追買，籌碼更健康。

**量測什麼**
今日融資餘額 - 昨日融資餘額 ≤ 0 → +10。餘額持平或下降都算符合。

**怎麼用**
這是免費版中非常有價值的反向確認。你希望看到：股票在漲，但融資沒有增加。融資持續增加的上漲要小心，代表散戶在追，一旦賣壓來了，融資斷頭會加速下跌。

---

### 2F-5. 三大法人同向 +5

**為什麼**
外資、投信、自營商同時淨買，是市場最強的共識信號。三個性質不同的機構（外資長線、投信中線、自營短線）都判斷今天偏多，意見一致的概率相對低，一旦同向代表強度。

**量測什麼**
外資淨買 > 0 且投信淨買 > 0 且自營商淨買 > 0 → 額外 +5（在各自 +15/+10/+5 基礎上）。

**怎麼用**
這是三法人各自得分之外的加成分。如果三法人都買，基本分已有 +30，再加 +5 = +35，Pillar 2 幾乎滿分。搭配外資連買（2F-6）使用：連買多日且三法人同向，是最高品質的籌碼信號。

---

### 2F-6. 外資連買天數 +5

**為什麼**
單日外資淨買可能是再平衡或對沖需要，不代表真正看多。連續多日外資淨買，才代表有持倉建立意圖。連買3日以上，幾乎可排除技術性買盤的可能。

**量測什麼**
從今日往前回看，連續幾日外資淨買（含今日）≥ 3 → +5。系統最多回看14個日曆日（約7個交易日），跳過非交易日。

**怎麼用**
連買天數越多，信號越強，但分數上限是5分（不隨天數遞增）。連買10天和連買3天都是+5。更長的連買天數體現在 LLM 推理文字中，不在分數裡。

---

### 2F-7. 投信連買天數 +5（新）

**為什麼**
理由與外資連買（2F-6）相同，但針對投信。台灣投信有明顯的「月底作帳」行為——月末3-4日的買進可能只是淨值作帳，不代表真實看法。連買3日以上，穿越月底作帳的短期雜訊，才是真建倉信號。

**量測什麼**
與外資連買的計算邏輯完全一致：用現有的 T86 回看資料提取投信列，連續淨買 ≥ 3日 → +5。實作上零新增網路請求（T86 已快取）。

**怎麼用**
投信連買搭配外資連買同時出現，是台股中比較罕見的強信號。兩個都達到3日連買 → 共 +10，代表法人同步建倉。

---

### 2F-8. 自營商連買天數 +3（新）

**為什麼**
自營商是短線操盤，連買3日代表他們認為短線趨勢向上。自營商不像投信有作帳壓力，連買行為更純粹是市場判斷。分數設定為+3（低於投信+5），因為自營商持倉周期最短，連買行為的持續性預測能力相對較低。

**量測什麼**
連續自營商淨買 ≥ 3 日 → +3。同樣使用 T86 回看資料，零新增請求。

**怎麼用**
自營商連買是輔助確認，非主導因子。三大法人都連買時，整體免費版籌碼分數可達到高水位。

---

### 2F-9. 融資使用率 +5 / -5（新）

**為什麼**
融資使用率 = 融資餘額 / 融資限額（TWSE 核定的最高可融資張數）。使用率低（< 20%），代表市場對這隻股票的融資興趣不高，散戶沒有追漲，籌碼乾淨；使用率高（> 80%），代表融資接近額度上限，散戶槓桿已接近飽和，一旦跌勢開始，強制平倉風險很高。

**量測什麼**
- 使用率 < 20% → +5（健康，法人主導）
- 使用率 20%–80% → 0（中性，不加不減）
- 使用率 > 80% → -5（危險，融資飽和）

需要 MI_MARGN 揭露`融資限額`欄位（需驗證欄位存在）。

**怎麼用**
這個因子特別適合用在進場前的最後確認：一隻各方面都強的股票，如果融資使用率超過80%，進場風險顯著增加。反過來，同樣強的股票融資使用率低，是更安全的進場時機。

---

### 2F-10. 借券賣出占比 -5（新）

**為什麼**
借券（SBL, Securities Borrowing and Lending）是機構放空的主要工具（不同於融券散戶放空）。外資和避險基金用借券賣出建立空頭部位，不影響融券餘額統計，所以如果只看融券，會漏掉機構空頭。借券賣出占當日成交比重超過10%，代表有機構在積極做空。

**量測什麼**
TWSE TWT93U 揭露每日借券賣出成交量。借券賣出股數 / 當日總成交股數 > 10% → -5。

**注意**：此因子需要先執行 `python3 scripts/validate_sbl_endpoint.py` 確認 TWSE 端點可用（目前環境因 IP 限制，TWSE 部分端點無法連線）。

**怎麼用**
借券賣出是機構的聰明空頭。當你看到一隻股票各指標都強，但借券占比超過10%，代表有人在對賭這個漲勢。這是一個反向信號，扣分合理。搭配融券數據：融券大（散戶空頭）+ 借券大（機構空頭），雙重空頭壓力。

---

## Pillar 3 — 空間（max 45 pts）

### 3-1. 20日高點突破 +20

**為什麼**
月線高點（約20個交易日）是技術分析中最常被機構參考的短期阻力位。突破月線高點，代表突破了過去一個月所有買進者的成本區——沒有套牢的賣壓。空間打開，後續阻力少。

**量測什麼**
今日收盤 ≥ 最近20日（含今日）最高價 × 0.99（允許1%的誤差，容納接近但未完全突破的情形）。

**怎麼用**
這是 Pillar 3 最重要的20分。沒有這分，即使動能和籌碼都強，可能只是在箱型震盪中的上緣。結合量能突破（1-2）使用：「量增突破月高」是技術分析中的經典買點。

---

### 3-2. 60日高點突破 +10

**為什麼**
60日高點（約3個月）是季線概念。突破季線高點，代表突破了一個完整財報季的多頭阻力。這比突破月線高點更有意義：機構在等季報，季線突破通常伴隨基本面催化劑。

**量測什麼**
今日收盤 ≥ 最近60日最高價 × 0.99。需要至少40個交易日的歷史資料（否則跳過並標記 `INSUFFICIENT_HISTORY_60D`）。系統拉取95個日曆日的 OHLCV 確保足夠資料。

**怎麼用**
60日突破是比20日突破更強的信號（分數反而低是因為它更少見，不能作為基礎要求）。同時達到20日和60日突破 → +30，是 Pillar 3 最強的組合。

---

### 3-3. 均線多頭排列 +5

**為什麼**
MA5 > MA10 > MA20 是均線「多頭排列」的定義。這代表5日、10日、20日均線依序遞減，說明短中長期買家的平均成本都是上升的，趨勢是整齊向上的。

**量測什麼**
今日 MA5（5日收盤均價）> MA10 > MA20，需要至少20個交易日歷史。

**怎麼用**
多頭排列是趨勢健康度的確認，不是信號觸發因素。在突破信號中，有多頭排列的比沒有的更可信——代表趨勢已經持續一段時間，不是剛開始。如果均線仍在糾纏或反轉（空頭排列），突破成功率較低。

---

### 3-4. MA20 斜率向上 +5

**為什麼**
均線多頭排列說明「當下位置」，MA20 斜率說明「趨勢方向」。MA20 斜率為正，代表20日均線本身在上揚，趨勢在加速中（而不是剛好排列好但即將走平）。

**量測什麼**
比較今日 MA20 值與5個交易日前的 MA20 值，若今日 > 5日前 → 斜率為正 → +5。需要至少25個交易日（MA窗口20 + 比較窗口5）。

**怎麼用**
斜率向上是趨勢強度的確認。MA20 斜率為正 + 多頭排列 = 趨勢不只正確而且在加速。如果多頭排列但 MA20 斜率趨平，趨勢可能接近末段。

---

### 3-5. RS vs 大盤 +5

**為什麼**
「個股強於大盤」是選股最基本的要求。如果大盤今天漲5%，你的股票只漲1%，即使各項技術指標都好，相對表現差代表資金沒有特別偏好這隻股票。RS 確保系統不是在大盤強漲時買入落後股。

**量測什麼**
比較個股5日報酬（今日收盤 / 5日前收盤 - 1）與加權指數（TAIEX）5日報酬。個股報酬 > TAIEX 報酬 × 1.2 → +5。乘以1.2代表至少要強過大盤20%，不是略高就算。

**怎麼用**
這分很有用：如果一隻股票各指標都強，但最近5天大盤漲了10%而它只漲了8%，RS 不過關。這可能代表資金正在輪動離開這個族群。反之，大盤橫盤而個股獨漲，RS 過關，是強者恆強信號。

## 風險扣分

### R-1. 隔日沖在 Top3 -25（付費版）

這是系統中最嚴重的扣分，也是設計上最重要的護城河。隔日沖主力的行為模式是：大量買進（製造籌碼集中假象）→ 隔日高開賣出（讓追進的散戶接貨）。如果系統偵測到這個模式，整個信號立刻被破壞。

-25 的扣分幅度是刻意的：即使其他指標都很強（動能 55 + 空間 45），一旦有隔日沖，信號就會被壓到 CAUTION 區間。

**實際意義**：看到 `隔日沖_TOP3` flag，不要進場。無論其他數字多漂亮。

---

### R-2. 融券暴增且券資比 > 15% -10

融券代表放空的散戶。**融券暴增**（今日融券餘額 > 昨日 × 1.2，即單日增加超過20%）且**券資比超過15%**（放空的比融資買進的多），雙重條件觸發才扣分。

這代表市場上有顯著的空頭力量，且是快速增加的新空頭。雖然有時空頭回補是多頭燃料，但在進場時機上，大量新增空頭是不確定性信號。

---

## LLM 分析輔助指標（不計分）

以下指標不影響信心分數，但會呈現在 StrategistAgent 的文字推理中：

| 指標 | 說明 | 使用方式 |
|------|------|---------|
| RSI(14) | 相對強弱指數 | >70 超買（追高風險）；<30 超賣（但確認底部需其他信號） |
| MACD 線/訊號線 | 動能振盪指標 | golden cross（黃金交叉）是中期看多；dead cross 是看空 |
| MA20 連續站上天數 | 正數=站上幾天，負數=跌破幾天 | 連續站上 >10 天，趨勢健康；剛剛站上，需要確認 |
| 跳空缺口% | 今日開盤相對昨收的缺口 | 跳空下跌 >1% 是當日警示信號 |
| 距52週高點% | 收盤距52週高點的百分比距離 | 距高點 <5%，可能面臨壓力；創新高，突破意義更大 |
| 當沖占比（新） | 當日當沖成交/總成交 | >30% 代表短線投機氣氛濃；>50% 的股票隔日容易回落 |
| 融券回補天數（新） | 融券餘額 / 日均成交量（天） | >3 天代表空頭回補需要多個交易日，軋空潛力存在 |

## 信號判讀門檻

| 模式 | LONG（進場） | WATCH（觀察） | CAUTION（迴避） |
|------|-------------|--------------|----------------|
| 付費版 | ≥ 70 | 40–69 | ≤ 30 |
| 免費版 | ≥ 60 | 35–59 | ≤ 30 |
| 例外 | 免費版且籌碼分 = 0 → 強制 CAUTION（TWSE API 無資料） | | |

**最高理論分數：**
- 付費版：P1(55) + P2_paid(45) + P3(45) = 145，扣掉風險最低 0 = 145。實際上限 100（系統 clamp）。
- 免費版：P1(55) + P2_free(63) + P3(45) = 163，同樣 clamp 到 100。

分數意義：「機會品質」不是「會不會漲」。分數100的股票也可能因為外部因素（總體環境、突發事件）下跌。這個系統的功能是在統計上篩出勝率較高的進場時機，不是水晶球。

## 常見組合解讀

**最強信號（付費版）**
- VWAP +20、量能突破 +20、連漲 +5 → P1 = 45+
- 淨買家數差 +15、集中度 +15、無隔日沖 +10、FII 在場 +5 → P2 = 45
- 月高突破 +20、季高突破 +10、多頭排列 +5 → P3 = 35+
- 總分 → clamp 到 100，LONG，信心極高

**需要警覺的高分場景**
- 分數 80+，但 flags 有 `隔日沖_TOP3` → 已自動扣 25，實際分數重新計算
- 分數 65，但 flags 有 `TWSE:TWSE_T86_ERROR` → 籌碼資料缺失，免費版 chip_pts = 0 → 強制 CAUTION
- 分數 58（免費版），但融資使用率 -5 把分數壓到 53 → WATCH，不是 LONG

**只有動能、無籌碼確認**
- P1 滿分 55 + P3 滿分 45 = 100，但 P2 = 0 → 免費版強制 CAUTION（chip_pts = 0 guard）
- 這個設計是刻意的：純技術面強、籌碼完全不明，進場風險不可量化
