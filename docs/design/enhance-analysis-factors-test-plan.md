## Edge Cases

- TWSE API network failure вҶ’ `ChipProxyFetcher` returns zero-value `TWSEChipProxy`, no exception raised
- TWSE API returns partial data (еӨ–иіҮ present, иһҚиіҮ missing) вҶ’ partial scoring, flag added
- `free_tier_mode=True` + both TWSE proxies pass + Pillar1+3 pass вҶ’ score = 85, LONG signal valid
- MA20 calculated from exactly 24 sessions (minimum) вҶ’ valid slope computed
- MA20 calculated from 23 sessions вҶ’ None slope, `insufficient_history_ma20` flag
- `_AnalysisHints.rsi` = 72 (overbought) вҶ’ flag propagates to reasoning hints, score unchanged
- All hints None вҶ’ no hints section in LLM prompt, no error
